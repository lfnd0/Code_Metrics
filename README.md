# Code Metrics

  ## Instalação do ambiente virtual:
    python -m venv env
  
  ## Ativação do ambiente virtual:
    env\Scripts\activate
  
  ## Instalação Radon:
    pip install radon
  
  ## Instalação dependências:
    pip install -r requirements.txt
